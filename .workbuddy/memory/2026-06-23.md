### Vercel 永久部署 ✅
- 用户电脑无全局 Node.js/npx/brew → 指导安装 Node.js LTS (nodejs.org)
- npx vercel 登录失败（Vercel.com 被墙，TLS 连接断开）
- 改用网页版 Vercel 部署方案：
  - GitHub 创建 time-echo 公开仓库
  - 创建 GitHub Personal Access Token (ghp_eahM...) 用于推送认证
  - 终端需设代理才能连 GitHub：export https_proxy=http://127.0.0.1:7897
  - 用户使用 SakuraCat(Clash) 代理工具，端口 7897
  - git push 成功推到 GitHub
- Vercel 网页端 Import Git 仓库：zhangfugui0117-stack/time-echo
- 配置 Environment Variable：SILICONFLOW_API_KEY=sk-bdg...（硅基流动免费模型）
- 部署成功！永久网址：**https://time-echo-ivory.vercel.app**
- Vercel 账号：fugui12138
